# VaultFlow Frontend

The frontend is a Vue 3 + Vite control plane for authentication, uploads,
share-link management, metrics, and audit visibility.

```bash
npm install
npm run dev
```

Default local URL: `http://localhost:8081`

Environment variables:

- `VITE_PROXY_TARGET` for the dev-server API proxy
- `VITE_API_BASE_URL` for direct browser-to-API calls

See the root [README](../README.md) for the
full architecture, demo flow, design decisions, and scaling roadmap.
